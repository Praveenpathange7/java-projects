
# Guessing Number Game

This is a simple Java console application where the user tries to guess a randomly generated number between 1 and 100.
The program provides feedback after each guess, indicating whether the guess is too high or too low,
and counts the number of attempts taken to guess the correct number.

## Features
- Random number generation between 1 and 100
- User input via console
- Feedback for each guess (too high/too low)
- Tracks number of attempts
- Displays success message and stats when guessed correctly

## How to Run
1. Clone this repository or copy the `Guessing_Number.java` file.
2. Compile the Java file:
   ```bash
   javac Guessing_Number.java
   ```
3. Run the program:
   ```bash
   java Guessing_Number.java
   ```

## Example Output
```

50
Your guess is too high
Enter the number between 1 to 100 :
25
Your guess is too low
...
--- You Win ---
```
